# 简介

## 解锁时启动计时脚本

系统右下搜索：任务计划程序  

选【创建任务】  

![创建任务](./fig/1.png)

输入任务名称、配置Window 10  

![创建任务](./fig/2.png)

触发器：新建--选择【工作站解锁时】  

![工作站解锁时](./fig/3.png)

新建--浏览-选中要启动的程序或脚本  

起始于：父文件夹  

![起始于](./fig/4.png)
